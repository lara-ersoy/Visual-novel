# Mijn projectvoorstel

## 1. Het doel mijn van project
Mijn project wil ik een fantasie game worden, dat zich afspeelt in de middeleeuwen. Voor de game wil ik ideeën ophalen uit DND-achtige games, zoals de combat en de klassen-setting. De game wilde ik eerst als een visual novel creëren. Daarna kwam ik op het idee (na overleg op dinsdag) om de game in 2D-setting te maken, met een wereldmap, waarbij gemanoeuvreerd kan worden naar andere kamers. Wel wil ik  ook de optie openhouden voor een 3D-setting, in een live-omgeving die zich bevindt in een wereld. De game wil ik op de PC toegankelijk maken, om meer ruimte te hebben voor de interface van de game. Ook wil een combat-systeem ik opzetten. De verhaallijn heb ik in richtlijnen opgezet en zal ik verder uitleggen bij de vragen over mijn project. 

---

## 2. Het formaat
### 2.1 Het probleem 
Mijn project heeft als doel om de DND-achtige games meer toegankelijk te maken voor een breder publiek, sinds DND-achtige games normaliter een kleinere (niche) doelgroep heeft. DND-achtige games hebben vaak een systeem dat voor het normale publiek wat moeilijker wordt gevonden, en games zoals Baldur's Gate 3 hebben het makkelijker en aantrekkelijker gemaakt voor de normale gamer of zelfs de normale internet-gebruiker om complexere games te spelen.
### 2.2 Het doelgroep
Dit project is vooral gericht op jongvolwassenen, die familiar zijn met het internet, en daarmee ook gamen. Het is niet specifiek gericht op DND-fanaten of reguliere gamers, sinds ik de game wil proberen voor een breder publiek aantrekkelijk & toegankelijk te maken. 
### 2.3 De setting
De setting van de game is op de PC én laptop. Dit wil ik, omdat veel vergelijkbare uitgebreide games zoals de Witcher of Baldur's Gate alleen toegankelijk zijn op de PC zijn. Het probleem dat daarbij komt kijken, is dat game PC's starten vaak vanaf €1000,-. Hierbij gaat de prijs zelfs steeds hoger gaat naarmate je een een betere grafische kaart wil (voor het runnen van de games überhaupt) of meer ruimte. Games zoals de Sims maken veel winst, omdat de games opzettelijk steeds simpeler worden gemaakt opdat ze juist de laptop te spelen (met doorgaans een zwakke grafische kaart) zijn. Hierdoor hebben ze ook een bredere publiek, waarvoor hun games toegankelijk zijn. Daarnaast is een interface op PC en/of laptop beter dan een interface op de telefoon, sinds er meer beweegruimte is voor het spelen van de game.
### 2.4 Wat doet mijn oplossing beter?
Zoals in 2.3 eigenlijk al vermeld, zijn de meeste DND-achtige fantasy games alleen toegankelijk op de PC en niet op de normale laptop met een zwakke grafische kaart. Mijn doel met mijn game is dit soort games meer toegankelijk maken voor een breder doelgroep door deze DND-achtige games simplistischer te maken. Voor een persoon die nog nooit een game heeft gespeeld, moet mijn game begrijpbaar en toegankelijk zijn. Dit, in tegenstelling tot de meeste games in de fantasie-achtige thema die vaak (naar mijn mening) soms te complex worden gecreerd. Voor de gewone gamer is het makkelijk te begrijpen, maar niet voor iemand die niet familiair is ermee en kan hij of zij daardoor ook sneller afmaken.
